1. How to open

There are three ways to open Critic Browser:

① Open from a package and selecting all quality rules in the system by default.

② Open to select one or many packages from menu.

③ Open from the last configuration.

2. How to set configure

Selecting a list of software packages and a list of quality rules, this resulting on a new list of critiques that broke the selected rules.

3. How to use

There are three options to select on a critique

① Mark as wrong: If the selected critique is a false positive, mark as wrong it will delete the critique from the cache and it will be no more displayed.

② Transform: If the selected critique has the transform option, transform it will resolve automatically the critique.

③ Mark as To-Do: this option will mark the critique as a To-Do critique, as a reminder that it's necessary to fix later.
